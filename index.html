Discover the Benefits of CachyOS

Cutting-Edge Performance

Built on Arch Linux, CachyOS features the optimized linux-cachyos kernel utilizing the advanced BORE Scheduler for unparalleled performance.

Optimized Packages

Packages compiled with x86-64-v3, x86-64-v4 and Zen4 instruction sets with LTO. Core packages get PGO or BOLT optimization.

Multiple Desktop Environments

Choose from KDE Plasma, GNOME, XFCE, i3, Wayfire, LXQt, Openbox, Cinnamon, COSMIC, and many more during installation.

Flexible Installation

Choose between a user-friendly GUI installer based on Calamares or a streamlined CLI-based option.

Robust Kernel Support

Multiple scheduler options including BORE, EEVDF, sched-ext, ECHO, and RT. All kernels optimized for your CPU.
